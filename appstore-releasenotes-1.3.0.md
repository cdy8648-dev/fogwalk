# FogWalk 1.3.0 릴리즈노트 초안

## App Store "새로운 기능" (한국어 · 사용자용, 붙여넣기용)

```
이번 업데이트로 탐험이 더 풍성해졌어요.

• 나만의 장소, 새 아이콘 16종
  이모지 대신 앱 전용 일러스트 핀으로 — 맛집·카페·산책·낚시·캠핑까지
  내 장소를 더 예쁘게 표시해요. 기존에 남긴 장소도 자동으로 바뀝니다.

• 발견 도감을 4가지 테마로 정리
  건축·문화 / 자연 / 교통 / 기타 — 그동안 모은 랜드마크를
  한눈에 보기 좋게 새로 묶었어요.

• 해외에서도 지역별 달성률
  일본·미국·유럽 등 세계 곳곳에서 주·도·현 단위로 얼마나 밝혔는지
  달성률을 확인할 수 있어요. 방문한 나라의 지도 데이터를 자동으로
  준비합니다.

• 지역 정확도 개선
  과천·구리가 경기로, 세종이 세종으로 — 시/도 소속이 잘못 잡히던
  곳들을 바로잡았어요. 앱을 열면 과거 기록도 정확한 지역으로 다시
  집계됩니다.
```

## TestFlight "테스트 정보" (내부 검증 체크리스트)

```
[1.3.0 검증 항목]

1) 나만의 장소 — 지도/에디터/상세카드에 새 아이콘 16종이 뜨는지,
   예전에 이모지로 만든 장소가 아이콘으로 바뀌어 보이는지.

2) 발견 도감 — 컬렉션·발견 상세의 필터가 건축·문화/자연/교통/기타
   4개로 바뀌었는지, 각 분류 개수가 맞는지.

3) 여권(지역) — '경기'를 펼쳐 과천시·구리시가 들어있는지, '세종'이
   따로 잡히는지, 앱 첫 실행 후 과거 타일이 재집계되는지.

4) 해외 지역팩 — (가능하면) 해외 좌표에서 지역별 달성률이 뜨는지,
   미국/중국은 지도에 '데이터 받기' 칩이 뜨고 탭하면 받아지는지.

5) 배터리 — 백그라운드 주행/야간 정지 시 소모가 이전과 동일한지
   (핫패스 변경 없음, 회귀 없어야 정상).
```
