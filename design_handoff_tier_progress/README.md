# Handoff: Collection 뱃지 등급 승급 인터페이스 — 시안 2a (트랙 + 티어 헤더)

## Overview
FogWalk(RN/Expo) Collection 화면 뱃지 섹션 개편. 밝힌 칸 수에 따라 **10개 등급**, 각 등급마다 **4개 노드(목표 칸)**. 한 등급의 4노드를 모두 달성하면 다음 등급으로 승급. 채택안 **2a**: 기존 4노드 가로 트랙을 유지하고 위에 티어 헤더 + 10등급 핍을 얹은 발전형.

## About the Design Files
`FogWalk 등급 승급.dc.html`은 HTML 디자인 레퍼런스다 — **2a 섹션이 채택안** (2b·승급 모먼트는 참고). 그대로 복사하지 말고 기존 RN 패턴으로 재구현. 목업의 conic-gradient 링 게이지는 react-native-svg 원호로 그릴 것.

## Fidelity
High-fidelity — 색·크기·간격은 아래 스펙 기준. 등급 데이터(이름/노드 값)는 앱 데이터 소스 사용.

## Layout (뱃지 카드, 위→아래)
카드: bg `#151827`, border 1 `#232844`, radius 24, padding 18 (top 18 / bottom 20).

### 1. 티어 헤더 (row, gap 12)
- **레벨 링** 52×52: 원형 게이지 — 진행률만큼 라임 `#C8F560`, 나머지 `#232844` (등급 내 노드 달성 비율 = 달성 노드/4). 안쪽 4px inset에 `#0D0F1A` 원, 중앙에 "LV"(DM Mono 8, `#7C8294`) + 등급 숫자(Space Grotesk Bold 22, `#C8F560`).
- **이름 블록** (flex:1): 등급명 17/800 `#F4EFE6` + 스케일 감각 12 `#7C8294`, 면적 수치만 Space Grotesk 600 `#8B7CFF`.
- **10등급 핍** (우측 정렬): 10개 점 gap 3. 달성 등급 `#C8F560`(7×7), 현재 등급 9×9 + glow `0 0 8px #C8F560`, 미달 `#2E3450`. 아래 "N / 10" DM Mono 9 `#5B617A`.

### 2. 4노드 트랙 (marginTop 24, 좌우 padding 6)
- 배경 트랙 라인: 노드 중심 잇는 높이 4 `#232844`, radius 2. 그 위에 달성 구간만 `#C8F560` 오버레이(마지막 달성 노드까지 + 현재 진행분).
- 4개 노드 (justify space-between):
  - **달성**: 48원, 라임 클레이 `radial-gradient(circle at 34% 28%, #DDFF8E, #C8F560 62%, #A8D93E)`, ✓ 800 `#0D0F1A` 20, shadow `inset 0 2px 4px rgba(255,255,255,.5), 0 4px 10px rgba(200,245,96,.3)`.
  - **현재**: 52원, bg `#0D0F1A`, border 3 `#C8F560`, 🎯 22, 맥동 링 애니메이션(아래).
  - **잠금**: 48원, bg `#1A1E30`, border 1 `#2E3450`, 🔒 18, opacity .55.
  - 라벨(노드 아래 gap 8): Space Grotesk 700 13 — 달성/현재 강조는 `#C8F560`/`#F4EFE6`, 잠금 `#5B617A`. 값은 K/M 축약.

### 3. 진행 인셋 (marginTop 22)
- bg `#0D0F1A`, radius 16, padding 13/15.
- 상단 row: "다음 노드까지" 13 `#7C8294` ↔ "38.2K / 40K 칸" — 현재값 Space Grotesk 700 `#C8F560`, "/ 40K" Space Grotesk `#7C8294`, "칸" 시스템.
- 진행 바(marginTop 10): 높이 8, track `#232844` radius 4, fill `linear-gradient(90deg,#A8D93E,#C8F560)` = (현재값 − 이전노드) / (다음노드 − 이전노드).

### 4. 승급 힌트 (marginTop 13, row gap 8)
- 🏔️ + "마지막노드 달성 시 **LV.N+1 등급명** 승급" — 12 `#7C8294`, 등급명 700 `#8B7CFF`.
- 마지막 등급(LV.10)에서는 "최고 등급 달성" 류 문구로 대체.

## 애니메이션
- 현재 노드 맥동: `0%/100% box-shadow 0 0 0 0 rgba(200,245,96,.5); 50% 0 0 0 12px rgba(200,245,96,0)` 2.2s 무한. (RN: Reanimated withRepeat scale+opacity 링, 없으면 생략 가능.)

## 상태 판정 로직
```
tier = 밝힌칸이 속한 등급 (마지막 노드 미만이면 그 등급, 초과 시 다음 등급으로)
각 노드: 밝힌칸 >= 노드값 → 달성 / 아직 안 지난 첫 노드 → 현재 / 그 이후 → 잠금
링 게이지 = 달성 노드 수 / 4
```
- 4노드 모두 달성 시 자동 승급(다음 등급 헤더로 전환) + 승급 모먼트 오버레이 트리거(레퍼런스 하단 참조).
- 큰 수 축약: 1000+ K, 1,000,000+ M (소수 1자리, 예 1.1M).
