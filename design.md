# FogWalk 디자인 시스템

> "안개 속을 걸어 세상을 밝히는" 무드. 다크 베이스 + 네온 포인트.
> 밝힌 곳·보상은 빛나고, 미지는 어둡게 — **대비**가 핵심.

## 1. 원칙 — MZ · 핀터레스트(핀보드) 감성
- 다크 미니멀 베이스, 포인트 컬러는 의미가 있을 때만.
- **핀보드 무드**: 사진은 폴라로이드, 모으는 것들은 "붙여둔" 느낌. 살짝의 회전·그림자로 손맛.
- **격자를 깨라**: 똑같은 크기 카드를 줄세우는 "AI 대시보드" 느낌 금지. 메이슨리(2열, 높이 변주), 비대칭 강조, 가로 스크롤 등 리듬 주기.
- 둥근 모서리 크게(카드 20–22, 폴라로이드 4–6), 여백 넉넉히, 따뜻한 종이색 포인트.
- 정보 위계: 큰 숫자(성취) → 라벨(muted). 한국어 톤은 친근·간결.

## 4. 간격 · 모서리 · 그림자
- **간격**: 4의 배수 (4·8·10·12·16·20·24). 화면 패딩 16, 카드 패딩 16, gap 10–14.
- **모서리**: 카드 16 / 칩·뱃지 12–14 / 작은 요소 8–10 / FAB 원형 / pill 999.
- **그림자**: FAB·토스트만 — opacity 0.3, radius 6–10, offset (0,2~3).

## 5. 컴포넌트 패턴
- **Card**: surface + 1px border + radius 16 + padding 16.
- **StatTile**: 중앙정렬, 큰 숫자(accent) + 작은 라벨(muted).
- **FAB**: 원형 52, accent 배경(라임=위치/앰버=카메라), ink 아이콘, 배지 우상단.
- **Badge**: 정사각 카드. 해금=accent 테두리 + 컬러 이모지 / 잠금=border + opacity 0.6 + 🔒.
- **Celebration Toast**: surface + lime 테두리 + 이모지 + 타이틀(accent) + 서브(muted). spring 등장.
- **Marker**: 원형/이모지. 사진=라임 프레임, 랜드마크=보라(전설=앰버 테두리), 현재위치=라임 점+헤일로. 줌아웃 시 점으로.
- **Polaroid(사진)**: 흰 프레임 + 살짝 그림자/회전 + 하단 코멘트 영역(추후 좋아요·댓글).

## 6. 지도 스타일
- 목표: **아이보리 베이스(land) + POI·도로명 정리**, 물/공원/동네/산만 표시.
- 안개 2단계(near 옅음 / far 진함) 대비 유지.
- 베이스맵은 조용하게, 우리 마커(랜드마크·사진)가 주인공.

## 7. 모션
- 축하: `spring(friction 6)` + scale 0.8→1 + fade.
- (예정) 발견 시네마틱 리플레이, 레벨업 강조 연출.

## 8. 아이콘/이모지
- 카테고리·통계 이모지 일관 사용 (`constants/landmarks.ts` 등).
- UI 아이콘은 `@expo/vector-icons`로 통일 지향.
